# Sales Performance Dashboard | Microsoft Excel

## Project Overview
Developed an interactive Sales Performance Dashboard in Microsoft Excel to monitor sales performance across multiple cities and sales executives. The dashboard provides insights into top and bottom performers, target achievement, and performance gaps using Pivot Tables, Pivot Charts, KPIs, and slicers.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Formulas

## Dashboard Analysis
- City-wise Sales Filtering
- Top 5 Sales Executives
- Bottom 5 Sales Executives
- Target Achievement (%) Analysis
- Away From Target (%) Analysis
- Executive Performance Comparison

## Key Features
- Interactive city slicers for dynamic filtering.
- Identifies the top and bottom performing sales executives.
- Tracks target achievement percentage for each executive.
- Measures the gap between actual sales and target.
- Visualizes performance using column, pie, and line charts.
